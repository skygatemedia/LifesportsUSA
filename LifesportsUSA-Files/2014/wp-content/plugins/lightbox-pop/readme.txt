== Description ==

A quicklook into Lightbox Pop

    ★ Create popup lightbox 
    ★ Full control on popup content
    ★ Standard WordPress content editor
    ★ Display position and dimension control
    ★ Timeout based popup display
    ★ Display based on browsed number of pages
    ★ Css settings like color, z-index, overlay opacity, border etc
    ★ Shortcode for displaying popup in specific pages

This plugin allows you to create a lightbox popup with custom contents. You can customize the popup display by configuring various settings such as  position settings (height, width, top,left), display logic settings (time delay after page load, number of pages to browse, lightbox repeat interval) and style settings(z-index, overlay opacity, color, border etc). You can use the plugin to display any type of contents such as special promotions, subscription forms, social media icons, feedback forms, video presentations and much more.

★ [Lightbox Pop User Guide](http://docs.xyzscripts.com/wordpress-plugins/lightbox-pop/ "Lightbox Pop User Guide")

= 1. What is a "lightbox" ? =

A lightbox is used to display any content inside a popup within a webpage, while rest of the page is greyed out and is not clickable. You may also call it greybox or thickbox. It is normally used to show subscription forms, contact forms, promotions, etc

= 2. The Lightbox Pop plugin is not working properly =

Please check the wordpress version you are using. Make sure it meets the minimum version recommended by us. Make sure all files of the `lightbox pop` plugin are uploaded to the folder `wp-content/plugins/`

= 3. Can I use this plugin with other popup plugins ? =

You can use it with any other popup related plugins from XYZScripts. However it may interfere with popup plugins from other vendors.

= 4. Can I add my own styles for the overlay ? =

Yes, you can change the style of the overlay from the settings page. If you need to use an image for the overlay, you have to use the premium version.

= 5. Can I set repeat interval in minutes ? =

Yes, you can set repeat interval in hours as well as minutes.

= 6. What kind of contents can be placed in a popup? =

You may  use the popup to display any type of contents such as special promotions, subscription forms, feedback forms, social media icons, video presentations and much more.

= 7. Can I display lightbox popup in specific pages only ? =

Yes, you can do this by using the shortcode provided from settings page.

= 8. Can I create multiple lightboxes for different pages ? =

You need to purchase the premium version for having this feature.
